### Prerequisites

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Debanga-06/Project-Python-100.git
   cd Project-Python-100
   ```

2. **Navigate to any project**
   ```bash
   cd "Project - 01"
   ```

3. **Install dependencies** (if required)
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the project**
   ```bash
   # For easy version
   cd easy_version
   python main.py
   
   # For advanced version
   cd ../advanced_version
   python main.py
   ```
